Rationale

All types are always visible as compact pills — zero hidden state, zero dropdowns. A single tap switches type and the left border updates in sync. The inactive pills fade to near-invisible so they don't compete with field content, but remain discoverable. Best for users who frequently switch types mid-row.

Rationale

The toolbar compresses 5 types into 5 icon buttons (140px total) — much narrower than text pills. An emoji icon makes each type instantly scannable.
